# Tcl의 regexp: 정규 표현식 처리 명령어

## 개요
Tcl에서 `regexp`는 정규 표현식을 사용하여 문자열을 검색하고, 일치하는 패턴을 찾는 강력한 명령어입니다. 이 명령어는 문자열의 특정 부분을 추출하거나 변환하는 데 유용합니다.

## 문서화
`regexp` 명령어는 주어진 문자열에서 정규 표현식 패턴을 검색하여 일치 여부를 확인하고, 필요에 따라 일치하는 부분을 추출할 수 있습니다. 이 명령어는 다음과 같은 형식을 사용합니다:

```tcl
regexp ?옵션? 패턴 문자열 ?변수? ?서브패턴...?
```

### 매개변수 설명:
- **옵션**: 일치 검사를 조정하기 위한 선택적 플래그입니다. (예: `-nocase`는 대소문자를 구분하지 않음)
- **패턴**: 검색할 정규 표현식 패턴입니다.
- **문자열**: 패턴을 검색할 대상 문자열입니다.
- **변수**: (선택적) 일치한 부분 문자열을 저장할 변수입니다.
- **서브패턴**: (선택적) 추가적으로 추출하고 싶은 서브패턴을 지정할 수 있습니다.

### 사용 예시:
1. 기본적인 패턴 매칭
   ```tcl
   set text "Hello World"
   if {[regexp {Hello} $text]} {
       puts "패턴이 일치합니다."
   }
   ```

2. 서브패턴 추출
   ```tcl
   set text "Tcl version 8.6"
   set version ""
   if {[regexp {Tcl version (\d+\.\d+)} $text match $version]} {
       puts "버전: $version"
   }
   ```

## 설명
`regexp` 사용 시 주의할 점은 정규 표현식의 복잡성입니다. 간단한 패턴에는 문제가 없지만, 복잡한 패턴을 사용할 경우 예상치 못한 결과가 발생할 수 있습니다. 일반적인 실수는 다음과 같습니다:
- 대소문자를 구분하지 않도록 옵션을 설정하지 않음
- 패턴의 구문 오류
- 서브패턴의 수가 변수와 일치하지 않음

정규 표현식의 문법을 잘 이해하고 사용하는 것이 중요합니다. 또한, Tcl에서는 `regexp` 외에도 `regsub`, `regexp`, `regress` 같은 관련 명령어들이 존재하므로 필요에 따라 적절한 명령어를 선택해야 합니다.
