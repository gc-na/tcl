# Tcl의 "for" 루프: 반복문 사용법과 예제

## 개요
Tcl의 "for" 명령은 주어진 조건에 따라 코드 블록을 반복 실행하는 데 사용되는 강력한 반복문입니다. 이 명령은 주로 카운터 기반 루프에 적합하며, 특정 범위 내에서 반복을 수행할 수 있습니다.

## 문서화
### 목적
"for" 명령은 지정된 횟수만큼 코드 블록을 반복 실행할 수 있도록 합니다. 반복문은 코드의 효율성을 높이고, 반복적인 작업을 간단하게 처리하는 데 유용합니다.

### 사용법
"for" 명령의 기본 구조는 다음과 같습니다:

```tcl
for {initialization} {condition} {increment} {
    # 반복 실행할 코드
}
```

- **initialization**: 루프의 시작 부분에서 단 한번 실행되는 초기화 코드입니다. 일반적으로 변수의 초기값을 설정합니다.
- **condition**: 루프가 계속 실행될 조건입니다. 이 조건이 true일 경우, 루프 내부 코드가 실행됩니다.
- **increment**: 루프의 각 반복이 끝날 때마다 실행되는 코드입니다. 일반적으로 카운터 변수를 증가시키는 데 사용됩니다.

### 세부 사항
- "for" 루프는 주로 카운트 기반 반복에 사용되지만, 조건문과 함께 다른 형태의 반복도 구현할 수 있습니다.
- 루프의 조건이 false가 되면, 루프는 종료됩니다.
- 루프 내에서 사용할 수 있는 명령어는 Tcl의 모든 명령어와 동일합니다.

## 예제
### 기본 예제
다음은 0부터 4까지의 숫자를 출력하는 간단한 "for" 루프입니다.

```tcl
for {set i 0} {$i < 5} {incr i} {
    puts $i
}
```

### 역순 출력
다음 예제는 4부터 0까지 역순으로 출력합니다.

```tcl
for {set i 4} {$i >= 0} {incr i -1} {
    puts $i
}
```

## 설명
- **공통 오류**: "for" 루프의 조건이 항상 true인 경우, 무한 루프가 발생할 수 있습니다. 따라서 조건을 주의 깊게 설정해야 합니다.
- **가독성 문제**: 복잡한 초기화나 증가 로직은 가독성을 떨어뜨릴 수 있으므로, 가능한 간단하게 유지하는 것이 좋습니다.
- **변수 스코프**: "for" 루프 내에서 정의된 변수는 루프 외부에서도 접근할 수 있습니다. 이 점을 유의해야 합니다.

## 한 줄 요약
Tcl의 "for" 루프는 주어진 조건에 따라 반복적으로 코드 블록을 실행하는 강력한 반복문입니다.
