# Tcl의 switch 명령어: 조건에 따라 코드 블록 선택하기

## 개요
Tcl의 `switch` 명령어는 주어진 표현식의 값에 따라 여러 개의 코드 블록 중 하나를 선택하여 실행하는 제어 구조입니다. 이를 통해 코드의 가독성을 높이고, 복잡한 조건문을 간단하게 작성할 수 있습니다.

## 문서
### 목적
`switch` 명령어는 특정 조건에 따라 실행할 코드를 선택하는 데 사용됩니다. 여러 조건을 간결하게 표현할 수 있으며, 각각의 조건에 대해 특정 코드를 실행하도록 설계되었습니다.

### 사용법
`switch` 명령어의 기본 구문은 다음과 같습니다:

```tcl
switch ?옵션? 표현식 {
    값1 {
        ... (값1에 대한 코드)
    }
    값2 {
        ... (값2에 대한 코드)
    }
    ...
    default {
        ... (모든 경우에 해당하지 않을 때 실행할 코드)
    }
}
```

#### 옵션
- `-exact`: 값의 비교를 엄격하게 수행합니다. 기본값은 `-exact`입니다.
- `-glob`: 와일드카드 패턴을 사용하여 비교합니다.
- `-regexp`: 정규 표현식을 사용하여 비교합니다.

### 세부사항
- `switch` 명령어는 조건의 첫 번째 일치 항목을 찾고, 그에 해당하는 코드 블록을 실행합니다.
- `default` 블록은 모든 조건이 일치하지 않을 경우 실행됩니다.
- 코드 블록은 중괄호 `{}`로 감싸야 하며, 여러 줄의 코드를 포함할 수 있습니다.

## 예제
### 기본 사용 예제
```tcl
set fruit "apple"

switch $fruit {
    "apple" {
        puts "사과입니다."
    }
    "banana" {
        puts "바나나입니다."
    }
    default {
        puts "알 수 없는 과일입니다."
    }
}
```

이 예제에서는 `fruit` 변수가 `"apple"`일 때 "사과입니다."가 출력됩니다.

### glob 옵션 사용 예제
```tcl
set filename "report_2023.txt"

switch -glob $filename {
    "report_*.txt" {
        puts "텍스트 보고서 파일입니다."
    }
    default {
        puts "다른 유형의 파일입니다."
    }
}
```

위의 예제에서 `filename`이 `"report_2023.txt"`인 경우 "텍스트 보고서 파일입니다."가 출력됩니다.

## 설명
`switch` 명령어는 여러 조건을 효과적으로 처리할 수 있지만, 다음과 같은 일반적인 함정이 있습니다:
- 조건의 순서: 조건은 위에서 아래로 평가되므로, 더 일반적인 조건을 먼저 두지 않도록 주의해야 합니다.
- 문자열 비교: 기본적으로 `-exact`로 비교하므로, 값이 정확히 일치해야 합니다. 와일드카드나 정규 표현식을 사용하려면 적절한 옵션을 설정해야 합니다.

## 한 줄 요약
