# Tcl 배열 (Array) 사용법과 특징

## 개요
Tcl에서 배열(array)은 키-값 쌍으로 데이터를 저장하는 자료구조로, 동적으로 크기를 조정할 수 있는 유연한 구조입니다. 배열은 문자열 인덱스를 사용하여 데이터를 효율적으로 관리할 수 있습니다.

## 문서화
Tcl 배열은 데이터의 집합을 관리하는 데 유용하게 사용됩니다. 배열은 여러 개의 값을 하나의 변수에 그룹화하여 저장할 수 있게 해줍니다. 배열은 다음과 같은 주요 특징을 가지고 있습니다:

- **동적 크기 조정**: 배열은 필요에 따라 크기를 조정할 수 있으며, 초기에 크기를 지정할 필요가 없습니다.
- **키-값 쌍**: 배열은 문자열 키를 사용하여 값에 접근합니다. 각 키는 배열 내에서 고유해야 합니다.
- **다양한 데이터 유형**: 배열에는 문자열, 숫자 등 다양한 데이터 유형을 저장할 수 있습니다.

### 사용법
Tcl에서 배열을 사용할 때는 `array` 명령어를 사용합니다. 배열을 생성하고 조작하는 기본 구문은 다음과 같습니다:

1. 배열 생성: `array set arrayName {key1 value1 key2 value2 ...}`
2. 값 설정: `set arrayName(key) value`
3. 값 가져오기: `set value [set arrayName(key)]`
4. 배열 크기 확인: `array size arrayName`
5. 모든 키와 값 나열: `array names arrayName`, `array get arrayName`

## 예제
다음은 Tcl 배열을 사용하는 기본 예제입니다:

```tcl
# 배열 생성 및 초기화
array set myArray {name "홍길동" age 30 city "서울"}

# 배열 값 가져오기
puts "이름: $myArray(name)"
puts "나이: $myArray(age)"
puts "도시: $myArray(city)"

# 배열 값 수정
set myArray(age) 31

# 배열의 크기 확인
puts "배열의 크기: [array size myArray]"

# 모든 키와 값 출력
puts "모든 키: [array names myArray]"
puts "모든 값: [array get myArray]"
```

## 설명
Tcl 배열을 사용할 때 몇 가지 주의할 점이 있습니다:

- **키의 중복**: 배열 내의 모든 키는 고유해야 하므로 중복된 키를 추가하면 기존 값이 덮어씌워집니다.
- **사전형 데이터**: 배열은 사전형 데이터 구조로, 복잡한 데이터를 저장할 때 유용합니다. 그러나 배열의 키는 문자열만 허용되므로 숫자나 다른 타입의 변수를 키로 사용할 수 없습니다.
- **메모리 관리**: 배열은 메모리를 자동으로 관리하지만, 불필요한 배열을 제거할 때는 `unset arrayName`을 사용하여 메모리를 해제해야 합니다.

## 한 문장 요약
Tcl 배열은 동적 크기의 키-값 쌍을 관리할 수 있는 유연한 자료구조로, 문자열 인덱스를 사용하여 데이터를 저장하고 조회하는 데 유용합니다.
