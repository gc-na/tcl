# Tclのpwdコマンド: 現在の作業ディレクトリを取得する方法

## 概要
`pwd`コマンドは、Tclにおいて現在の作業ディレクトリを表示するための基本的なコマンドです。このコマンドを使用することで、スクリプトの実行中に現在のディレクトリを把握することができます。

## ドキュメント
### 目的
`pwd`コマンドは、現在の作業ディレクトリのパスを取得するために使用されます。この情報は、ファイル操作やディレクトリの管理において非常に重要です。特に、スクリプトがどのディレクトリで実行されているかを確認する際に役立ちます。

### 使用法
`pwd`コマンドは、特定の引数を必要とせず、単独で呼び出します。コマンドを実行すると、現在の作業ディレクトリの絶対パスが返されます。

#### 基本的な構文
```tcl
set currentDir [pwd]
```

### 詳細
- 出力は、スラッシュ（/）で区切られたパス形式で表示されます。
- `pwd`コマンドは、特定のファイルやディレクトリの存在を確認する際に、他のファイル操作コマンドと組み合わせて使用することが一般的です。

## 例
以下に、`pwd`コマンドの基本的な使用例を示します。

### 例1: 現在の作業ディレクトリを表示
```tcl
puts "現在の作業ディレクトリは: [pwd]"
```

### 例2: 現在の作業ディレクトリを変数に格納
```tcl
set currentDir [pwd]
puts "格納されたディレクトリ: $currentDir"
```

## 説明
`pwd`コマンドを使用する際の一般的な注意点：
- スクリプトが異なるディレクトリから実行されると、出力されるパスも異なります。スクリプトの実行前にカレントディレクトリを確認しておくことが重要です。
- `pwd`は、シェル環境によっては異なる動作をする場合がありますが、Tclでは一貫して現在の作業ディレクトリを返します。
