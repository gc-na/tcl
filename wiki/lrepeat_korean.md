# Tcl의 lrepeat: 리스트 반복 생성하기

## 개요
Tcl의 `lrepeat` 명령은 주어진 요소를 특정 횟수만큼 반복하여 새로운 리스트를 생성하는 데 사용됩니다. 이 명령은 리스트를 다루는 Tcl 프로그래밍에서 유용하게 활용됩니다.

## 문서
### 목적
`lrepeat` 명령은 주어진 요소를 여러 번 반복하여 리스트를 생성하는 기능을 제공합니다. 이를 통해 데이터 구조를 쉽게 구성할 수 있으며, 대량의 데이터를 처리할 때 유용합니다.

### 사용법
`lrepeat`의 기본 구문은 다음과 같습니다:

```tcl
lrepeat count element
```

- `count`: 반복할 횟수를 지정하는 정수입니다.
- `element`: 리스트에 추가할 요소입니다.

이 명령은 `count`가 0일 경우 빈 리스트를 반환합니다.

### 세부 사항
- `count`는 음수일 수 없으며, 0 이상의 정수여야 합니다.
- `element`는 단일 값일 수도 있고, 복합 데이터 타입(리스트, 문자열 등)일 수도 있습니다.
- 이 명령은 새로운 리스트를 생성하며, 원본 리스트를 변경하지 않습니다.

## 예제
아래는 `lrepeat`의 기본적인 사용 예입니다.

```tcl
# 숫자 5를 3번 반복
set result [lrepeat 3 5]
# 출력: {5 5 5}
puts $result

# 문자열 "abc"를 2번 반복
set result2 [lrepeat 2 "abc"]
# 출력: {abc abc}
puts $result2

# 빈 리스트 생성
set result3 [lrepeat 0 "xyz"]
# 출력: {}
puts $result3
```

## 설명
`lrepeat`를 사용할 때 주의해야 할 점:

- `count`가 음수인 경우 Tcl은 오류를 발생시킵니다.
- `element`가 리스트인 경우, 해당 리스트가 단일 요소로 간주되어 반복됩니다. 즉, 리스트의 각 요소가 개별적으로 반복되는 것이 아닙니다.
- 즉, `lrepeat 3 {a b}`를 실행하면 결과는 `{a b a b a b}`가 아니라 `{a b a b a b}`가 됩니다.

## 한 줄 요약
`lrepeat`는 특정 요소를 지정한 횟수만큼 반복하여 새로운 리스트를 생성하는 Tcl 명령입니다.
