# Tclの「return」コマンド：関数からの戻り値を返す方法

## 概要
Tclにおける「return」コマンドは、プロシージャから戻り値を返すために使用されます。このコマンドは、関数からの実行を終了し、指定した値を呼び出し元に返します。

## ドキュメンテーション
### 目的
「return」コマンドは、Tclのプロシージャ（関数）で実行されるコードの結果を呼び出し元に返し、プロシージャの実行を終了します。

### 使用法
```tcl
return ?value?
```
- `value`（省略可能）：戻り値として返したい値を指定します。値が指定されない場合、デフォルトで空の文字列が返されます。

### 詳細
- プロシージャの中で「return」を使用すると、そのプロシージャは即座に終了し、指定された値を呼び出し元に返します。
- 「return」コマンドは、プロシージャの最後でなくても呼び出すことができ、条件に応じて異なる値を返すことが可能です。
- プロシージャ外で「return」を使用することはできず、必ずプロシージャの内部で使用する必要があります。

## 例
### 基本的な使用例
```tcl
proc add {a b} {
    return [expr {$a + $b}]
}

set result [add 3 5]
puts $result  ; # 出力: 8
```

### 条件による戻り値
```tcl
proc check_even {num} {
    if { $num % 2 == 0 } {
        return "Even"
    } else {
        return "Odd"
    }
}

puts [check_even 4]  ; # 出力: Even
puts [check_even 3]  ; # 出力: Odd
```

## 説明
- **一般的な落とし穴**: プロシージャの外で「return」を使用するとエラーが発生します。必ずプロシージャ内部でのみ使用してください。
- **デフォルト値**: 戻り値を指定しない場合、空の文字列が返されることに留意してください。この動作は、呼び出し元での処理に影響を与える可能性があります。
- **複数の戻り値**: Tclは複数の戻り値をサポートしており、リストを返すことで複数の値を返すことができます。

## 一文要約
Tclの「return」コマンドは、プロシージャから戻り値を返し、実行を終了するために使用されます。
