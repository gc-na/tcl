# استخدام الأمر "case" في Tcl: دليل شامل

## ملخص
يعتبر الأمر "case" في Tcl أداة قوية لتنفيذ عمليات فرعية بناءً على تطابق قيمة معينة مع مجموعة من الخيارات المحددة. يوفر الأمر "case" وسيلة بسيطة وفعالة لتوجيه تدفق البرنامج بناءً على قيم متغيرة.

## الوثائق
### الغرض
يستخدم الأمر "case" في Tcl لتحديد مجموعة من الشروط التي تتوافق مع قيمة معينة. يتم استخدامه بشكل شائع في المواقف التي تحتاج فيها إلى تنفيذ إجراءات مختلفة استنادًا إلى قيمة متغيرة.

### الاستخدام
يتم استخدام الأمر "case" بالطريقة التالية:

```tcl
case expression {
    pattern1 { commands1 }
    pattern2 { commands2 }
    ...
    default { default_commands }
}
```

- **expression**: القيمة التي سيتم مقارنتها.
- **pattern**: الأنماط التي سيتم مطابقتها مع القيمة.
- **commands**: الأوامر التي سيتم تنفيذها إذا تطابقت القيمة مع النمط.

### التفاصيل
- يمكن استخدام الأنماط كقيم ثابتة أو تعبيرات تعكس مجموعة من القيم المحتملة.
- يمكن أن تحتوي كل مجموعة من الأوامر في حالة على أكثر من أمر واحد.
- إذا لم تتطابق أي من الأنماط، سيتم تنفيذ أوامر القسم "default" إذا تم تحديده.

## الأمثلة
### مثال بسيط
```tcl
set color "أحمر"

case $color {
    "أحمر" { puts "اللون هو أحمر" }
    "أخضر" { puts "اللون هو أخضر" }
    "أزرق" { puts "اللون هو أزرق" }
    default { puts "لون غير معروف" }
}
```

### مثال مع أنماط متعددة
```tcl
set fruit "تفاح"

case $fruit {
    "تفاح" - "موز" { puts "الفاكهة هي تفاح أو موز" }
    "برتقال" { puts "الفاكهة هي برتقال" }
    default { puts "فاكهة غير معروفة" }
}
```

## الشرح
### الأخطاء الشائعة
- **عدم استخدام القسم "default"**: في حال عدم تحديد القسم الافتراضي، قد يؤدي ذلك إلى عدم تنفيذ أي إجراءات إذا لم تتطابق القيمة المدخلة مع أي من الأنماط.
- **نسيان الأقواس**: يجب التأكد من وضع الأقواس بشكل صحيح حول الأوامر، وإلا ستظهر أخطاء في التفسير.

### ملاحظات إضافية
- يمكن استخدام التعبيرات العادية في الأنماط، مما يوفر مرونة أكبر في حالة المطابقة.
- يعتبر "case" خيارًا ممتازًا عند التعامل مع القيم المتعددة، بدلاً من استخدام عدة جمل if.

## ملخص من جملة واحدة
الأمر "case" في Tcl هو وسيلة فعالة لتنفيذ أوامر متفاوتة بناءً على تطابق قيمة معينة مع أنماط محددة.
