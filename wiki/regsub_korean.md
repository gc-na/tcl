# Tcl의 regsub 명령어: 정규 표현식으로 문자열 치환하기

## 개요
Tcl의 `regsub` 명령어는 정규 표현식을 사용하여 문자열 내에서 특정 패턴을 찾아 치환하는 기능을 제공합니다. 이 명령어는 텍스트 처리, 데이터 정제 및 다양한 문자열 조작에서 유용하게 사용됩니다.

## 문서화
### 목적
`regsub` 명령어는 주어진 문자열에서 정규 표현식에 일치하는 부분을 찾아 다른 문자열로 치환합니다. 이를 통해 복잡한 문자열 조작이 가능해집니다.

### 사용법
`regsub`의 기본 구문은 다음과 같습니다:
```tcl
regsub ?-all? exp string subst ?varName?
```
- `-all`: 선택 사항으로, 문자열 내의 모든 일치하는 부분을 치환합니다. 이 옵션이 없으면 첫 번째 일치하는 부분만 치환됩니다.
- `exp`: 치환할 정규 표현식입니다.
- `string`: 치환할 대상 문자열입니다.
- `subst`: 정규 표현식에 일치하는 부분을 대체할 문자열입니다.
- `varName`: (선택 사항) 결과를 저장할 변수의 이름입니다.

### 세부 사항
`regsub`는 문자열 내에서 정규 표현식을 통해 패턴을 찾고, 이를 지정된 문자열로 대체합니다. 대체 문자열 내에서 `\n` 형식을 사용하면 정규 표현식의 캡처 그룹을 참조할 수 있습니다. 예를 들어, `\1`, `\2`는 각각 첫 번째, 두 번째 캡처 그룹을 참조합니다.

## 예제
### 기본 사용 예제
```tcl
set str "Hello World"
set result [regsub {World} $str "Tcl"]
puts $result  ;# 출력: Hello Tcl
```

### 모든 일치 치환 예제
```tcl
set str "abc def abc ghi"
set result [regsub -all {abc} $str "XYZ"]
puts $result  ;# 출력: XYZ def XYZ ghi
```

### 캡처 그룹 사용 예제
```tcl
set str "Tcl is fun"
set result [regsub {(\w+) is} $str {\1는}]
puts $result  ;# 출력: Tcl는 fun
```

## 설명
`regsub` 사용 시 몇 가지 주의해야 할 점이 있습니다. 
- 정규 표현식의 문법을 정확히 이해하고 사용해야 합니다. 잘못된 표현식은 예상치 못한 결과를 초래할 수 있습니다.
- `-all` 옵션을 사용하지 않으면 첫 번째 일치하는 부분만 치환되므로, 필요에 따라 옵션을 적절히 설정해야 합니다.
- 대체 문자열 내에서 캡처 그룹을 참조할 때, 잘못된 인덱스를 입력하면 오류가 발생합니다.
