# Tcl의 expr: 수식 평가 및 계산

## 개요
Tcl의 `expr` 명령은 수학적 표현식을 평가하고 계산하는 데 사용됩니다. 이 명령은 정수 및 부동 소수점 연산을 지원하며, 수학적 계산을 통해 다양한 프로그래밍 작업을 수행할 수 있도록 돕습니다.

## 문서화
`expr` 명령은 수식이나 표현식을 인자로 받아 해당 수식의 값을 계산합니다. 주로 수학적 계산을 위해 사용되지만, 비교 연산 및 논리 연산도 수행할 수 있습니다. `expr`는 다음과 같은 형태로 사용됩니다:

```tcl
expr expression
```

### 목적
`expr`의 주 목적은 수학적 및 논리적 표현식을 평가하여 결과를 반환하는 것입니다. 이를 통해 프로그래머는 복잡한 계산을 쉽게 수행할 수 있습니다.

### 사용법
- `expr` 명령은 산술 연산자(`+`, `-`, `*`, `/`, `%`)와 비교 연산자(`==`, `!=`, `<`, `>`, `<=`, `>=`), 논리 연산자(`&&`, `||`, `!`)를 지원합니다.
- 표현식 내에서 괄호를 사용하여 연산의 우선 순위를 조정할 수 있습니다.

### 세부 정보
- `expr`은 기본적으로 부동 소수점 연산을 지원합니다. 정수 연산 시에도 결과는 부동 소수점으로 반환될 수 있습니다.
- `expr`은 Tcl의 다른 명령어와 함께 사용할 수 있으며, 변수와 함께 동적으로 표현식을 생성할 수 있습니다.

## 예제
### 기본 사용 예제
```tcl
set a 10
set b 20
set result [expr {$a + $b}]
puts "결과: $result"  ;# 결과: 30
```

### 비교 연산 예제
```tcl
set x 5
set y 10
if {[expr {$x < $y}]} {
    puts "$x는 $y보다 작습니다."
}
```

### 논리 연산 예제
```tcl
set isTrue 1
set isFalse 0
if {[expr {$isTrue && !$isFalse}]} {
    puts "조건이 참입니다."
}
```

## 설명
`expr` 사용 시 주의할 점은 다음과 같습니다:
- 표현식이 복잡해질수록 괄호를 적절히 사용하여 연산의 우선 순위를 명확히 해야 합니다.
- 변수의 값이 비어있거나 정의되지 않은 경우, `expr`에서 오류가 발생할 수 있으니, 변수를 사용하기 전에 반드시 정의했는지 확인해야 합니다.
- `expr`은 문자열과 숫자를 혼합하여 사용할 경우, 자동 타입 변환이 이루어지지만, 예상치 못한 결과를 초래할 수 있습니다.

## 요약
Tcl의 `expr` 명령은 수학적 및 논리적 표현식을 평가하여 결과를 반환하는 강력한 도구입니다.
