# Tcl의 case 명령어: 다중 조건 분기 처리 방법

## 개요
Tcl에서 `case` 명령어는 주어진 표현식의 값에 따라 여러 조건을 확인하고 그에 맞는 코드 블록을 실행하는 데 사용됩니다. `case`는 복잡한 조건 분기를 간결하게 처리할 수 있는 유용한 도구입니다.

## 문서
### 목적
`case` 명령어는 표현식의 결과에 따라 여러 가지 선택지를 정의하고, 각 선택지에 맞는 명령어를 실행할 수 있도록 도와줍니다. 이는 특정 조건에 따라 다른 행동을 취해야 할 때 매우 유용합니다.

### 사용법
`case` 명령어의 기본 구문은 다음과 같습니다:

```tcl
case expression {
    pattern1 { command1 }
    pattern2 { command2 }
    ...
    default { default_command }
}
```

- **expression**: 평가할 표현식입니다.
- **pattern**: 표현식의 결과와 비교할 패턴입니다.
- **command**: 해당 패턴이 일치할 경우 실행할 명령어입니다.
- **default**: 모든 패턴이 일치하지 않을 경우 실행할 명령어입니다 (선택적).

### 상세 설명
- `case` 명령어는 `switch` 명령어와 유사하지만, 주로 패턴 매칭에 중점을 두고 있습니다.
- 각 패턴은 쉼표로 구분하여 나열할 수 있으며, 여러 패턴이 일치할 경우 첫 번째 일치하는 패턴의 명령어가 실행됩니다.
- `default` 블록은 모든 패턴이 일치하지 않을 때 실행되는 기본 동작을 정의합니다.

## 예제
다음은 `case` 명령어의 기본 사용 예입니다.

```tcl
set fruit "사과"

case $fruit {
    "사과" { puts "사과를 선택했습니다." }
    "바나나" { puts "바나나를 선택했습니다." }
    "오렌지" { puts "오렌지를 선택했습니다." }
    default { puts "알 수 없는 과일입니다." }
}
```

이 코드는 변수 `fruit`의 값에 따라 해당하는 메시지를 출력합니다.

## 설명
- **일치하지 않는 패턴**: 주의할 점은, `case` 블록에 정의된 패턴과 표현식의 값이 일치하지 않을 경우 `default` 블록이 실행된다는 것입니다.
- **패턴 형태**: 패턴은 문자열 뿐만 아니라 와일드카드(`*`)를 사용할 수 있어 보다 유연한 조건 처리가 가능합니다.
- **빈 표현식**: 표현식이 비어있는 경우, `default` 블록이 실행됩니다.

## 한 줄 요약
Tcl의 `case` 명령어는 주어진 표현식의 값에 따라 여러 조건을 평가하고 해당하는 코드 블록을 실행하는 다중 분기 처리 도구입니다.
