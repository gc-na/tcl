# Tcl에서 lappend 명령어: 배열에 요소 추가하기

## 개요
Tcl의 `lappend` 명령어는 리스트에 새로운 요소를 추가하는 기능을 제공합니다. 이 명령어는 기존 리스트를 수정하거나 새로운 리스트를 생성할 때 유용합니다. `lappend`는 리스트의 끝에 요소를 추가하며, 리스트가 존재하지 않을 경우 자동으로 생성합니다.

## 문서화
### 목적
`lappend`는 Tcl에서 리스트에 요소를 추가하기 위해 사용됩니다. 이 명령어는 간단한 구문으로 리스트를 수정할 수 있도록 도와줍니다.

### 사용법
```tcl
lappend listName ?value value ...?
```
- **listName**: 요소를 추가할 리스트의 이름입니다.
- **value**: 리스트에 추가할 값입니다. 여러 개의 값을 쉼표로 구분하여 추가할 수 있습니다.

### 세부사항
- `lappend`는 지정된 리스트가 존재하지 않으면 자동으로 새 리스트를 생성합니다.
- 리스트가 존재할 경우, 주어진 값들이 리스트의 끝에 추가됩니다.
- `lappend`는 리스트를 직접 수정하므로, 반환 값이 없으며 리스트의 상태가 변경됩니다.

## 예제
### 기본 사용 예제
```tcl
set myList {1 2 3}
lappend myList 4
puts $myList  ;# 출력: 1 2 3 4
```

### 여러 값 추가하기
```tcl
lappend myList 5 6 7
puts $myList  ;# 출력: 1 2 3 4 5 6 7
```

### 리스트가 없는 경우
```tcl
lappend newList 10
puts $newList  ;# 출력: 10
```

## 설명
`lappend`를 사용할 때 주의해야 할 점은 다음과 같습니다:
- 리스트가 비어있거나 존재하지 않을 경우, `lappend`를 호출하면 새로운 리스트가 생성되므로, 원하지 않는 결과가 발생할 수 있습니다.
- `lappend`는 원본 리스트를 직접 수정하기 때문에, 원본 리스트의 상태를 유지하고 싶다면, 리스트를 복사한 후에 사용해야 합니다.
- 리스트에 추가할 값이 문자열일 경우, 공백을 포함한 문자열은 하나의 요소로 취급됩니다. 예를 들어, `lappend myList "hello world"`는 리스트에 `hello world`라는 단일 요소를 추가합니다.

## 한 줄 요약
