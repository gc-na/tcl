# Tcl에서의 proc: 함수 정의와 호출의 모든 것

## 개요
Tcl에서 `proc` 명령은 사용자 정의 프로시저(함수)를 생성하는 데 사용됩니다. 이를 통해 코드의 재사용성을 높이고, 복잡한 작업을 단순화할 수 있습니다.

## 문서화
### 목적
`proc` 명령은 Tcl 스크립트 내에서 프로시저를 정의하는 데 사용됩니다. 프로시저는 특정 작업을 수행하는 코드 블록으로, 필요할 때 여러 번 호출할 수 있습니다.

### 사용법
`proc` 명령의 기본 구문은 다음과 같습니다:

```tcl
proc 프로시저명 {인자1 인자2 ...} {
    명령어들
}
```

- **프로시저명**: 생성할 프로시저의 이름입니다.
- **인자**: 프로시저가 받을 입력값(매개변수)입니다.
- **명령어들**: 프로시저가 수행할 Tcl 명령어의 집합입니다.

### 세부 사항
- 프로시저를 정의할 때 인자는 중괄호 `{}`로 감싸야 합니다.
- 프로시저 내에서 `return` 명령을 사용하여 값을 반환할 수 있습니다.
- 프로시저는 다른 프로시저 내에서 호출할 수 있으며, 재귀 호출도 가능합니다.
- 프로시저가 정의된 후에는 언제든지 호출할 수 있습니다.

## 예제
### 기본 사용 예
다음은 간단한 프로시저를 정의하고 호출하는 예입니다.

```tcl
proc helloWorld {} {
    puts "Hello, World!"
}

helloWorld
```

이 코드는 "Hello, World!"를 출력합니다.

### 인자가 있는 프로시저 예
인자를 사용하는 프로시저의 예는 다음과 같습니다.

```tcl
proc addNumbers {a b} {
    return [expr {$a + $b}]
}

set result [addNumbers 5 7]
puts "Result: $result"
```

이 코드는 "Result: 12"를 출력합니다.

## 설명
- 프로시저 이름은 고유해야 하며, 이미 정의된 프로시저와 이름이 겹치지 않도록 주의해야 합니다.
- 인자의 수는 정의된 대로 맞춰야 하며, 필요 시 기본값을 설정할 수 있습니다.
- 프로시저가 호출된 후, 정의된 명령어가 실행되며, `return` 문이 있을 경우 해당 값을 반환합니다.

## 한 줄 요약
Tcl의 `proc` 명령은 사용자 정의 프로시저를 생성하여 코드의 재사용성과 가독성을 높이는 기능입니다.
