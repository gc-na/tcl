# استخدام الأمر "foreach" في Tcl: دليل شامل

## ملخص
يعتبر الأمر "foreach" في Tcl من الأوامر الأساسية التي تسمح بتكرار تنفيذ كود معين على عناصر قائمة معينة. يسهل هذا الأمر التعامل مع المجموعات ويساعد في تحسين كفاءة كتابة الأكواد.

## الوثائق
### الغرض
الأمر "foreach" يُستخدم لتكرار تنفيذ مجموعة من الأوامر على عناصر معينة في قائمة. يمكن استخدامه للتعامل مع القوائم البسيطة أو القوائم المعقدة.

### الاستخدام
صيغة الأمر "foreach" هي كما يلي:

```tcl
foreach variable list {
    # الأوامر المراد تنفيذها
}
```

- **variable**: هو اسم المتغير الذي سيتم تعيين كل عنصر من عناصر القائمة عليه في كل تكرار.
- **list**: هي القائمة التي تحتوي على العناصر التي سيتم التكرار عليها.

### التفاصيل
- إذا كانت القائمة تحتوي على أكثر من عنصر، سيتم تنفيذ الأوامر داخل القوسين لكل عنصر على حدة.
- يمكنك أيضًا استخدام عدة متغيرات في الأمر "foreach" لتكرار تنفيذ الأوامر على قوائم متعددة.

## أمثلة
### مثال 1: استخدام قائمة بسيطة
```tcl
set fruits {تفاح برتقال موز}
foreach fruit $fruits {
    puts "أنا أحب $fruit"
}
```
**الناتج:**
```
أنا أحب تفاح
أنا أحب برتقال
أنا أحب موز
```

### مثال 2: استخدام متغيرات متعددة
```tcl
set names {أحمد علي سارة}
set scores {90 85 88}
foreach name $names score $scores {
    puts "$name حصل على $score"
}
```
**الناتج:**
```
أحمد حصل على 90
علي حصل على 85
سارة حصلت على 88
```

## الشرح
### الأخطاء الشائعة
- **نسيان الأقواس**: يجب التأكد من وضع الأوامر داخل الأقواس بعد "foreach".
- **عدم استخدام قائمة صحيحة**: إذا كانت القائمة فارغة، فلن يتم تنفيذ الأوامر الموجودة داخل "foreach".
- **استخدام متغيرات غير معرفة**: تأكد من تعريف المتغيرات المستخدمة في الأمر قبل استخدامها.

### ملاحظات إضافية
- يمكن استخدام "foreach" لتكرار عناصر القوائم المتداخلة، مما يوفر مرونة كبيرة في التعامل مع البيانات المركبة.
- يعتبر "foreach" أكثر كفاءة من استخدام حلقات "for" التقليدية في العديد من الحالات، نظرًا لأنه يتجنب الحاجة إلى إدارة عدد التكرارات يدويًا.

## ملخص جملة واحدة
الأمر "foreach" في Tcl يوفر وسيلة فعالة لتكرار تنفيذ الأوامر على عناصر قائمة معينة، مما يسهل التعامل مع البيانات في البرامج.
