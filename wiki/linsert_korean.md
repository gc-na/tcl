# Tcl의 linsert 명령어: 리스트 삽입하기

## 개요
Tcl의 `linsert` 명령어는 리스트의 지정된 인덱스 위치에 새로운 요소를 삽입하는 데 사용됩니다. 이 명령어를 통해 기존 리스트를 수정하고, 원하지 않는 위치에 새로운 값을 추가할 수 있습니다.

## 문서화
### 목적
`linsert` 명령어는 Tcl 리스트의 특정 위치에 하나 이상의 요소를 추가하여 새로운 리스트를 생성합니다. 원래 리스트는 변경되지 않으며, 변경된 리스트를 반환합니다.

### 사용법
```tcl
linsert list index element1 ?element2 ...?
```
- `list`: 요소를 삽입할 원래 리스트입니다.
- `index`: 요소를 삽입할 위치의 인덱스입니다. 인덱스는 0부터 시작하며, 음수 인덱스를 사용하면 리스트의 끝에서부터 계산할 수 있습니다.
- `element1`, `element2`, ...: 삽입할 요소들입니다. 하나 이상의 요소를 제공할 수 있습니다.

### 세부사항
- `linsert`는 원래 리스트를 변경하지 않고, 새로운 리스트를 반환합니다.
- 인덱스가 리스트의 범위를 초과하는 경우, 새로운 요소는 리스트의 끝에 추가됩니다.
- 음수 인덱스를 사용하여 리스트의 끝에서부터 요소를 삽입할 수 있습니다.

## 예제
### 기본 사용 예제
```tcl
set myList {a b c d}
set newList [linsert myList 2 x]
puts $newList  ;# 결과: a b x c d
```

### 여러 요소 삽입
```tcl
set myList {a b c d}
set newList [linsert myList 1 x y z]
puts $newList  ;# 결과: a x y z b c d
```

### 음수 인덱스 사용
```tcl
set myList {a b c d}
set newList [linsert myList -1 x]
puts $newList  ;# 결과: a b c x d
```

## 설명
`linsert` 명령어를 사용할 때 주의해야 할 몇 가지 사항이 있습니다:
- 인덱스가 리스트의 길이를 초과할 경우, 요소가 리스트의 끝에 추가됩니다. 이는 종종 예상치 못한 결과를 초래할 수 있습니다.
- 음수 인덱스를 사용할 경우, 리스트의 끝에서부터 계산되므로, 리스트의 길이를 정확히 알고 사용하는 것이 좋습니다.
- `linsert`는 여러 요소를 동시에 삽입할 수 있지만, 삽입하는 요소의 순서가 중요합니다. 지정된 인덱스에서부터 순서대로 삽입됩니다.
