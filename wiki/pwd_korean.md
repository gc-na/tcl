# Tcl의 pwd 명령어: 현재 작업 디렉토리 확인하기

## 개요
`pwd`는 Tcl 프로그래밍 언어에서 현재 작업 디렉토리를 출력하는 명령어입니다. 사용자나 프로그램이 현재 위치를 확인할 수 있도록 돕습니다.

## 문서화
### 목적
`pwd` 명령어는 현재 스크립트가 실행되고 있는 디렉토리의 절대 경로를 반환합니다. 이 명령어는 파일 작업이나 경로 설정을 할 때 유용하게 사용됩니다.

### 사용법
```tcl
pwd
```
- **반환값**: 현재 작업 디렉토리의 절대 경로 문자열.

### 상세 설명
- `pwd`는 "print working directory"의 약자로, 현재 작업 중인 디렉토리의 경로를 표시합니다.
- Tcl의 모든 스크립트는 기본적으로 특정 디렉토리에서 실행되며, 이 디렉토리를 확인할 필요가 있을 때 `pwd`를 사용합니다.
- 예를 들어, 파일을 생성하거나 수정하고자 할 때, 현재 디렉토리를 알면 경로 문제를 피할 수 있습니다.

## 예시
### 기본 사용 예
```tcl
set currentDir [pwd]
puts "현재 작업 디렉토리: $currentDir"
```
이 코드는 현재 작업 디렉토리를 가져와 출력합니다.

### 파일 경로 설정 예
```tcl
set filePath [file join [pwd] "example.txt"]
puts "파일 경로: $filePath"
```
위 코드는 현재 디렉토리와 파일 이름을 결합하여 완전한 파일 경로를 생성합니다.

## 설명
- **일반적인 함정**: `pwd`는 스크립트가 실행되는 디렉토리를 반환하므로, 스크립트가 다른 디렉토리로 전환된 경우, 예상과 다른 결과를 반환할 수 있습니다. 스크립트 실행 중에 `cd` 명령어로 작업 디렉토리를 변경했다면, `pwd`는 새로운 디렉토리를 반영합니다.
- **주의 사항**: `pwd` 명령어는 파일 시스템의 상태에 따라 다르게 작동할 수 있으며, 네트워크 드라이브나 가상 환경에서는 경로가 다르게 나타날 수 있습니다. 

## 한 줄 요약
