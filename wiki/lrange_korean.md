# Tcl의 lrange: 리스트에서 특정 인덱스 범위를 추출하는 명령어

## 개요
`lrange`는 Tcl 프로그래밍 언어에서 리스트의 특정 범위를 추출하는 데 사용되는 명령어입니다. 이 명령어는 리스트에서 시작 인덱스와 끝 인덱스 사이의 요소를 반환합니다. 

## 문서화
### 목적
`lrange` 명령어는 주어진 리스트에서 지정된 두 인덱스 사이의 요소를 잘라내어 새로운 리스트를 만드는 데 사용됩니다. 이는 리스트 데이터 조작에서 매우 유용한 기능입니다.

### 사용법
```tcl
lrange list start end
```
- **list**: 요소를 포함하는 리스트입니다.
- **start**: 추출을 시작할 인덱스입니다. 인덱스는 0부터 시작합니다.
- **end**: 추출을 끝낼 인덱스입니다. 이 인덱스의 요소는 포함됩니다. 

### 상세 설명
- 인덱스는 0부터 시작하며, 음수 인덱스를 사용할 경우 리스트의 끝에서부터 요소를 참조합니다. 예를 들어, -1은 리스트의 마지막 요소를 의미합니다.
- 만약 `start` 인덱스가 `end` 인덱스보다 크거나, 리스트의 범위를 벗어난 경우 빈 리스트가 반환됩니다.

## 예제
### 기본 사용 예제
```tcl
set myList {a b c d e f}
set result [lrange $myList 1 3]
puts $result  ; # 출력: b c d
```

### 음수 인덱스 사용 예제
```tcl
set myList {a b c d e f}
set result [lrange $myList -3 -1]
puts $result  ; # 출력: d e f
```

### 범위를 벗어난 예제
```tcl
set myList {a b c d e f}
set result [lrange $myList 4 10]
puts $result  ; # 출력: e f
```

## 설명
`lrange`를 사용할 때 주의해야 할 점은 인덱스의 범위와 리스트의 크기입니다. 만약 `start`가 `end`보다 크면 빈 리스트가 반환되며, 인덱스가 리스트의 길이를 초과할 경우에도 비어 있는 리스트가 반환됩니다. 음수 인덱스를 사용할 경우, 리스트의 길이에 따라 올바른 요소를 참조하는지 확인해야 합니다.
