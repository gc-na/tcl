# Tcl의 "cd" 명령어: 디렉토리 변경 방법

## 개요
Tcl에서 "cd" 명령어는 현재 작업 디렉토리를 변경하는 데 사용됩니다. 이 명령어는 Tcl 스크립트나 대화형 세션에서 파일 시스템의 특정 디렉토리로 이동하고자 할 때 유용합니다.

## 문서화
### 목적
"cd" 명령어는 Tcl에서 현재 작업 디렉토리를 설정하여 파일 및 디렉토리를 쉽게 관리할 수 있도록 합니다. 이 명령어를 사용하면 스크립트 내에서 파일 경로를 상대적으로 사용할 수 있습니다.

### 사용법
```tcl
cd ?directory?
```
- `directory`: 이동하고자 하는 목적지 디렉토리의 경로를 지정합니다. 이 인자는 생략할 수 있으며, 이 경우 부모 디렉토리로 이동합니다.

### 세부사항
- "cd" 명령어는 현재 작업 디렉토리를 변경하는 것 외에도, 작업 디렉토리가 변경된 후에는 상대 경로를 사용하여 파일에 접근할 수 있게 해줍니다.
- 만약 지정한 디렉토리가 존재하지 않거나 접근할 수 없는 경우, Tcl은 오류를 발생시킵니다.

## 예제
1. 기본 사용법:
   ```tcl
   cd /home/user/documents
   ```
   위 명령어는 `/home/user/documents` 디렉토리로 이동합니다.

2. 부모 디렉토리로 이동:
   ```tcl
   cd ..
   ```
   이 명령어는 현재 디렉토리의 부모 디렉토리로 이동합니다.

3. 현재 디렉토리 확인:
   ```tcl
   puts [pwd]
   ```
   `pwd` 명령어를 사용하여 현재 작업 디렉토리를 출력합니다.

## 설명
- **경로 문제**: 디렉토리를 변경할 때 경로가 올바르지 않으면 오류가 발생합니다. 따라서 경로를 정확하게 입력하는 것이 중요합니다.
- **상대 경로 사용**: 현재 디렉토리를 변경한 후에는 상대 경로를 사용하여 파일을 쉽게 접근할 수 있습니다.
- **스크립트 내 사용**: 스크립트 내에서 "cd"를 사용하여 파일 작업을 수행할 때, 경로가 변경되므로 이후 작업에 영향을 미칠 수 있습니다.

## 한 줄 요약
Tcl의 "cd" 명령어는 현재 작업 디렉토리를 변경하여 파일 시스템 내의 다른 위치로 이동할 수 있는 기능을 제공합니다.
