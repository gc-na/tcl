# Tcl의 lmap: 리스트를 변환하는 유용한 명령어

## 개요
Tcl 언어에서 `lmap`은 리스트의 각 요소에 대해 주어진 변환 작업을 수행하고, 그 결과를 새로운 리스트로 반환하는 강력한 명령어입니다. 이 명령어는 특히 리스트를 효율적으로 처리하고 변환할 때 유용합니다.

## 문서
### 목적
`lmap`은 주어진 리스트의 각 요소를 변경하는데 사용됩니다. 함수형 프로그래밍의 원리를 적용하여 각 요소에 대해 지정된 스크립트를 실행하고, 결과를 새로운 리스트로 생성합니다.

### 사용법
```tcl
lmap varName listName script
```
- `varName`: 리스트의 각 요소를 저장할 변수 이름입니다.
- `listName`: 변환할 원본 리스트입니다.
- `script`: 각 요소에 대해 실행할 Tcl 스크립트입니다.

### 세부 사항
- `lmap`은 원본 리스트를 변경하지 않고, 변환된 요소들을 포함하는 새로운 리스트를 반환합니다.
- `script` 내에서 `varName`을 사용하여 리스트의 각 요소에 접근할 수 있습니다.
- `lmap`은 여러 개의 리스트를 동시에 처리할 수 있으며, 이 경우 각 리스트의 요소는 인덱스에 따라 매칭됩니다.

## 예제
### 기본 사용 예제
```tcl
set myList {1 2 3 4 5}
set result [lmap x $myList {expr {$x * 2}}]
# result는 {2 4 6 8 10}입니다.
```

### 여러 리스트 처리 예제
```tcl
set list1 {1 2 3}
set list2 {4 5 6}
set result [lmap {a b} [list $list1 $list2] {expr {$a + $b}}]
# result는 {5 7 9}입니다.
```

## 설명
`lmap` 사용 시 주의해야 할 점은 다음과 같습니다:
- `lmap`은 원본 리스트를 수정하지 않으므로, 결과를 저장하는 변수를 따로 지정해야 합니다.
- `script`에서 사용된 변수는 각 리스트의 요소를 나타내므로, 변수가 다른 리스트의 요소를 참조할 경우 혼동을 피하기 위해 주의해야 합니다.
- 만약 리스트의 길이가 다르면, 짧은 리스트의 길이에 맞춰 결과가 생성됩니다. 이 점을 고려하여 리스트를 정의해야 합니다.

## 한 줄 요약
`lmap`은 Tcl에서 리스트의 각 요소를 변환하여 새로운 리스트를 생성하는 명령어입니다.
