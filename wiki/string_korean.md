# Tcl 문자열(string) 기능에 대한 완벽 가이드

## 개요
Tcl에서 문자열(string)은 텍스트 데이터의 기본 단위로, 문자열 조작을 위한 다양한 내장 명령어와 기능을 제공합니다. 이 문서에서는 Tcl의 문자열 처리 기능에 대한 목적, 사용법, 예제 및 주의사항을 다룹니다.

## 문서화
Tcl에서 문자열은 모든 유형의 텍스트 데이터를 표현할 수 있는 기본적인 데이터 타입입니다. 문자열은 다양한 형식의 데이터를 처리할 수 있는 유연성을 제공하며, 프로그래밍과 스크립팅에서 매우 중요합니다.

### 목적
Tcl의 문자열 기능은 텍스트 데이터의 생성, 조작 및 비교를 용이하게 하여 데이터 처리 및 분석 작업을 효율적으로 수행할 수 있도록 돕습니다.

### 사용법
Tcl에서 문자열을 사용하려면 다음과 같은 기본 명령어를 사용할 수 있습니다:

- `string`: 다양한 문자열 관련 작업을 수행하는 명령어입니다.
- `string length`: 문자열의 길이를 반환합니다.
- `string index`: 문자열의 특정 인덱스에 있는 문자를 반환합니다.
- `string compare`: 두 문자열을 비교합니다.

#### 기본 구문
```tcl
string command args
```

## 예제
1. 문자열 길이 구하기
   ```tcl
   set myString "Hello, Tcl!"
   set length [string length $myString]
   puts "문자열 길이: $length"  ; # 출력: 문자열 길이: 12
   ```

2. 문자열 인덱스 사용
   ```tcl
   set myString "Hello, Tcl!"
   set char [string index $myString 7]
   puts "인덱스 7의 문자: $char"  ; # 출력: 인덱스 7의 문자: T
   ```

3. 문자열 비교
   ```tcl
   set str1 "apple"
   set str2 "banana"
   set result [string compare $str1 $str2]
   puts "비교 결과: $result"  ; # 출력: 비교 결과: -1 (str1 < str2)
   ```

## 설명
Tcl의 문자열 처리에는 몇 가지 주의할 점이 있습니다. 

- **문자 인덱스**: Tcl은 0부터 시작하는 인덱스를 사용하므로 첫 번째 문자는 인덱스 0입니다.
- **대소문자 구분**: 문자열 비교는 대소문자를 구분하므로, "hello"와 "Hello"는 다르게 처리됩니다.
- **공백 처리**: 문자열의 앞뒤에 있는 공백은 문자열 길이에 포함되므로 주의해야 합니다.
