# Tcl 命令：pwd - 獲取當前工作目錄

## 概要
`pwd` 是 Tcl 語言中的一個命令，用於獲取當前工作目錄的完整路徑。這對於檔案操作和目錄管理非常有用。

## 文檔
### 目的
`pwd` 命令的主要目的是返回當前工作目錄的路徑，這對於在腳本中進行文件操作時確定當前上下文非常重要。

### 使用方法
```
pwd
```

### 詳細說明
- 當執行 `pwd` 命令時，它會返回一個字符串，該字符串代表當前工作目錄的絕對路徑。
- `pwd` 通常在需要確認當前目錄的情況下使用，特別是在執行涉及文件路徑的操作時。

## 示例
以下是一些基本的使用示例：

### 示例 1：獲取當前工作目錄
```tcl
set currentDir [pwd]
puts "當前工作目錄是: $currentDir"
```

### 示例 2：在腳本中使用
```tcl
proc showCurrentDir {} {
    puts "當前工作目錄是: [pwd]"
}

showCurrentDir
```

## 解釋
- **常見陷阱**：在使用 `pwd` 時，確保在正確的上下文中執行命令。如果在子進程中執行，可能會得到不同的工作目錄。
- **注意事項**：`pwd` 只會返回當前工作目錄的路徑，而不會改變它。如果需要更改當前工作目錄，可以使用 `cd` 命令。

## 一句話總結
`pwd` 命令在 Tcl 中用於獲取當前工作目錄的完整路徑。
