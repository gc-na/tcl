# Tcl의 foreach 명령어: 배열 및 리스트 반복 처리

## 개요
Tcl에서 `foreach`는 리스트나 배열의 각 요소를 반복적으로 처리할 수 있는 강력한 명령어입니다. 이 명령어는 주로 반복적인 작업을 간결하고 효율적으로 수행하는 데 사용됩니다.

## 문서화
`foreach` 명령어는 주어진 리스트의 각 요소에 대해 반복적으로 특정 작업을 수행하도록 설계되었습니다. 기본 구문은 다음과 같습니다:

```tcl
foreach variable list {
    # 실행할 코드
}
```

### 목적
`foreach`의 주된 목적은 리스트의 모든 요소를 순회하며 각 요소에 대해 코드를 실행하는 것입니다. 이는 특히 데이터 처리 및 배열 작업에서 유용합니다.

### 사용법
- **변수**: 반복할 요소를 저장할 변수입니다.
- **리스트**: 반복할 요소들의 집합입니다.
- **코드 블록**: 각 요소에 대해 실행할 Tcl 코드입니다.

리스트와 배열을 처리하는 데 효과적이며, 각 요소에 대한 인덱스를 자동으로 관리합니다.

## 예제
### 리스트 반복 예제
```tcl
set myList {apple banana cherry}
foreach fruit $myList {
    puts "Fruit: $fruit"
}
```
위 코드는 리스트의 각 과일 이름을 출력합니다.

### 배열 반복 예제
```tcl
set myArray(apple) 1
set myArray(banana) 2
set myArray(cherry) 3

foreach {key value} [array get myArray] {
    puts "$key: $value"
}
```
이 코드는 배열의 모든 키와 값을 출력합니다.

## 설명
`foreach`를 사용할 때 주의할 점은 다음과 같습니다:
- **변수 충돌**: 반복 변수 이름이 다른 코드에서 사용되는 변수와 충돌하지 않도록 주의해야 합니다.
- **리스트 형식**: 리스트가 비어 있을 경우 `foreach` 블록은 실행되지 않습니다.
- **중첩 사용**: `foreach`를 중첩하여 사용할 수 있지만, 코드 가독성이 떨어질 수 있으므로 주의가 필요합니다.

## 한 줄 요약
Tcl의 `foreach` 명령어는 리스트나 배열의 각 요소를 간편하게 반복 처리하는 데 유용한 기능입니다.
