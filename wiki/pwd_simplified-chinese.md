# Tcl 中的 pwd 命令详解

## 概述
`pwd` 是 Tcl 编程语言中的一个内置命令，用于获取当前工作目录的绝对路径。它是“print working directory”的缩写，广泛用于文件操作和路径管理。

## 文档
### 目的
`pwd` 命令用于返回当前工作目录的完整路径。这在需要进行文件读取、写入或其他文件系统操作时非常有用。

### 用法
`pwd` 命令的基本语法如下：
```tcl
pwd
```

### 详细说明
- 当调用 `pwd` 时，它将返回一个字符串，表示当前的工作目录。
- 该命令不接受任何参数，也不具有选项。
- `pwd` 通常与 `cd` 命令一起使用，以便在切换目录后确认当前目录的变化。

## 示例
以下是 `pwd` 命令的基本使用示例：

1. **获取当前工作目录**
   ```tcl
   set currentDir [pwd]
   puts "当前工作目录是: $currentDir"
   ```

2. **在切换目录后验证**
   ```tcl
   cd /tmp
   puts "切换到的目录是: [pwd]"
   ```

## 解释
- **常见陷阱**：在某些情况下，用户可能会误解 `pwd` 与 `cd` 命令的作用。`pwd` 只用于显示当前目录，而不会更改工作目录。
- **注意事项**：如果在执行 `pwd` 之前没有使用 `cd` 命令改变目录，返回的将是 Tcl 启动时的工作目录。
- **平台差异**：在不同操作系统上，返回的路径格式可能会有所不同（例如，Windows 使用反斜杠 `\`，而 Unix/Linux 使用正斜杠 `/`）。

## 一句话总结
`pwd` 命令用于获取并返回当前工作目录的绝对路径，是 Tcl 文件操作中的基础命令。
