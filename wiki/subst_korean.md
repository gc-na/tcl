# Tcl의 subst 명령어: 문자열 대체의 모든 것

## 개요
Tcl의 `subst` 명령어는 문자열 내에서 변수를 확장하고, 표현식을 계산하여 최종 결과를 생성하는 데 사용됩니다. 이 명령어는 문자열 처리와 텍스트 조작에 필수적인 도구로, 특히 동적으로 생성된 문자열을 다룰 때 유용합니다.

## 문서화
### 목적
`subst` 명령어는 Tcl의 문자열을 처리할 때 변수와 표현식을 평가하여 최종적으로 수정된 문자열을 반환하는 데 사용됩니다. 이를 통해 사용자 정의 변수나 명령어의 결과를 문자열 내에서 쉽게 활용할 수 있습니다.

### 사용법
`subst` 명령어의 기본 구문은 다음과 같습니다:
```tcl
subst ?-nocomplain? string
```
- `-nocomplain`: (선택 사항) 이 옵션이 주어지면, 오류가 발생할 경우 무시하고 원래 문자열을 반환합니다.
- `string`: 변환할 문자열입니다.

`subst`는 문자열 내의 변수를 확장하고, 명령어를 실행하여 그 결과를 문자열로 변환합니다. 예를 들어, `$variable`이나 `[command]`와 같은 표현식을 포함할 수 있습니다.

### 세부사항
- `subst`는 문자열 내 모든 변수를 확장합니다. 이때, 변수가 정의되지 않으면 빈 문자열로 처리됩니다.
- 중첩된 표현식도 지원하며, 이를 통해 복잡한 문자열 조작이 가능합니다.
- `subst`는 문자열을 평가한 후, 이를 반환합니다. 이 과정에서 원본 문자열은 변경되지 않습니다.

## 예제
### 기본 사용 예제
```tcl
set name "Tcl"
set greeting "Hello, $name!"
puts [subst $greeting]
```
위의 예제에서 `subst`는 `greeting` 문자열 내의 `$name` 변수를 확장하여 `Hello, Tcl!`을 출력합니다.

### 명령어 평가 예제
```tcl
set result [subst {The sum of 2 and 3 is [expr 2 + 3]}]
puts $result
```
이 경우, `expr` 명령어가 먼저 실행되어 결과적으로 `The sum of 2 and 3 is 5`가 출력됩니다.

## 설명
`subst`를 사용할 때 주의해야 할 몇 가지 사항이 있습니다:
- 변수가 정의되지 않은 경우, `subst`는 빈 문자열을 반환합니다. 따라서, 변수의 존재 여부를 확인하는 것이 좋습니다.
- 문자열 내에서 중괄호 `{}`와 대괄호 `[]`를 사용할 때, 평가 순서에 유의해야 합니다. 중괄호는 문자열로 취급되고, 대괄호는 Tcl 명령어로 해석됩니다.
- `subst`는 문자열을 반환하기 때문에, 이 결과를 반드시 변수에 저장하거나 출력해야 합니다.

## 한 줄 요약
Tcl의 `subst` 명령어는 문자열 내의 변수를 확장하고 표현식을 평가하여 최종 결과 문자열을 생성하는 데 사용됩니다.
