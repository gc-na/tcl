# Tcl의 "if" 명령어: 조건문 사용법 및 예제

## 개요
Tcl 프로그래밍 언어에서 "if" 명령어는 조건에 따라 코드 블록을 실행할 수 있도록 해주는 제어 구조입니다. 이는 프로그래밍 로직을 구성하는 데 있어 필수적인 요소로, 다양한 조건을 평가하고 그 결과에 따라 다른 작업을 수행하는 데 유용합니다.

## 문서화
### 목적
"if" 명령어는 주어진 조건이 참인지 거짓인지에 따라 특정 코드 블록을 실행하도록 합니다. 이를 통해 조건부 로직을 구현할 수 있습니다.

### 사용법
"if" 명령어의 기본 구문은 다음과 같습니다:

```tcl
if {조건} {
    # 조건이 참일 때 실행할 코드
} else {
    # 조건이 거짓일 때 실행할 코드 (선택 사항)
}
```

- **조건**: 평가할 조건식을 포함하는 중괄호로 감싸진 표현식.
- **실행할 코드**: 조건에 따라 실행될 Tcl 명령어들로, 중괄호로 감싸야 합니다.

### 세부사항
- "if" 명령어는 중첩(Nested)될 수 있으며, 여러 "else if" 구문을 추가하여 다중 조건을 처리할 수 있습니다.
- 조건식이 참일 경우, 해당 블록의 코드가 실행되며, 이후의 "else" 또는 "else if" 블록은 무시됩니다.
- "if" 문은 항상 조건식을 중괄호로 감싸야 하며, 조건식 내에서 불리언 연산자(예: `==`, `!=`, `<`, `>`, `&&`, `||`)를 사용할 수 있습니다.

## 예제
### 기본 사용 예제
```tcl
set num 10

if {$num > 5} {
    puts "num은 5보다 큽니다."
} else {
    puts "num은 5보다 작거나 같습니다."
}
```

### 여러 조건 처리
```tcl
set score 85

if {$score >= 90} {
    puts "A학점입니다."
} elseif {$score >= 80} {
    puts "B학점입니다."
} elseif {$score >= 70} {
    puts "C학점입니다."
} else {
    puts "D학점 이하입니다."
}
```

## 설명
- "if" 조건에서 중괄호({})를 사용하지 않으면 Tcl이 조건을 제대로 평가하지 못할 수 있습니다.
- 여러 "elseif" 구문을 사용할 때는 각각의 조건을 명확하게 구분해야 하며, 마지막 "else"는 선택 사항입니다.
- 코드 블록 내에서 변수나 명령어를 사용할 때에는 올바른 범위를 고려해야 하며, 예상치 못한 결과를 방지하기 위해 디버깅을 해야 할 수 있습니다.

## 한 줄 요약
