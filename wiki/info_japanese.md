# Tclの「info」コマンド: 機能と使い方

## 概要
Tclの「info」コマンドは、スクリプトの実行環境に関する情報を取得するための強力なツールです。これにより、変数、コマンド、パッケージなどの状態を確認することができます。

## ドキュメンテーション
「info」コマンドは、さまざまなオプションを持ち、特定の情報を取得するために使用されます。主な用途は、スクリプト内のデータや環境に関する情報を取得することです。

### 使用法
基本的な構文は以下の通りです：

```tcl
info option ?arg arg ...?
```

### 主なオプション
- `vars`: 現在のスコープ内のすべての変数名のリストを返します。
- `commands`: 現在のスコープ内のすべてのコマンド名のリストを返します。
- `packages`: 現在ロードされているパッケージの情報を返します。
- `tclversion`: 現在のTclインタプリタのバージョンを返します。

## 例
以下に「info」コマンドの基本的な使用例を示します。

### 例1: 変数のリストを取得
```tcl
set a 1
set b 2
puts [info vars]
```
出力:
```
a b
```

### 例2: コマンドのリストを取得
```tcl
puts [info commands]
```
出力:
```
info puts set ...
```

### 例3: 現在のTclバージョンを取得
```tcl
puts [info tclversion]
```
出力:
```
8.6
```

## 説明
「info」コマンドは非常に便利ですが、いくつかの注意点があります。特に、スコープの違いにより、変数やコマンドが見つからない場合があります。また、特定のオプションを使用する際には、引数を正しく指定する必要があります。注意深く使用することで、デバッグやスクリプトの最適化に役立つ情報を得ることができます。
