# Tcl에서 "incr" 명령어: 변수 값 증가시키기

## 개요
Tcl 프로그래밍 언어에서 "incr" 명령어는 특정 변수의 값을 간편하게 증가시키는 데 사용됩니다. 이 명령어는 주로 카운터를 관리하거나 반복문에서 값을 조정할 때 유용합니다.

## 문서화
### 목적
"incr" 명령어는 변수의 값을 지정된 양만큼 증가시키는데 사용됩니다. 이 명령어는 숫자 타입의 변수에만 적용되며, 기본적으로 1만큼 증가시키는 기능을 제공합니다.

### 사용법
```
incr 변수명 ?증가량?
```
- **변수명**: 증가시킬 변수의 이름.
- **증가량**: (선택 사항) 변수의 값을 얼마나 증가시킬지를 지정하는 정수. 기본값은 1입니다.

### 세부 사항
- "incr"는 변수의 값이 숫자형일 때만 작동합니다. 만약 비숫자형 변수에 사용하면 오류가 발생합니다.
- 변수는 자동으로 생성되며, 존재하지 않는 변수에 대해 "incr"를 호출하면 해당 변수가 0으로 초기화된 후 증가됩니다.

## 예제
### 기본 사용 예
```tcl
set counter 5
incr counter
puts $counter  ;# 출력: 6
```

### 증가량 지정 예
```tcl
set score 10
incr score 5
puts $score  ;# 출력: 15
```

### 음수 증가 예
```tcl
set balance 100
incr balance -20
puts $balance  ;# 출력: 80
```

## 설명
- **일반적인 실수**: "incr"를 사용할 때 변수의 초기 값이 숫자가 아닐 경우 오류가 발생합니다. 예를 들어, 문자열이나 리스트에 대해 "incr"를 호출하면 Tcl은 오류를 반환합니다.
- **변수 초기화**: 존재하지 않는 변수를 "incr"로 증가시키면, Tcl은 그 변수를 0으로 초기화한 후 증가시키게 됩니다.
- **스코프**: "incr"는 현재 스코프에서 변수에 접근하므로, 전역 변수를 증가시키려면 `global` 키워드를 사용해야 합니다.
